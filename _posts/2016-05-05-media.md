今天遇到一个问题，简单描述就是：

	调试好的页面在浏览器打印的时候样式丢失

检查后发现引用 css 是这样写的：

	<link href="path" rel="stylesheet" type="text/css" media="screen" />

去掉 media 属性，问题解决。

那么，这个 media 属性是做什么的呢？简单说就是：

	media 属性规定目标 URL 是为什么类型的媒介/设备进行优化的

	screen：计算机屏幕
	print：打印预览模式/打印页面
	all：默认，适合所有设备

[具体请点击](http://www.w3school.com.cn/html5/att_a_media.asp)
