## clone

1. 默认会把仓库中的所有分支都复制到本地

	git clone

2. 只克隆指定的某个分支，例如： git clone -b 指定的分支名字

	git clone -b &lt;branch&gt; &lt;remote_repo&gt;

## commit

1. 修改已经提交的注释（没有push过的）

	git commit --amend

## push

1. 将本地分支推送到远程新分支上

	git push origin &lt;remote_branch_name&gt;
